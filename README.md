🏥 CareMetrics Hospital Analytics Dashboard

Project Title / Headline
Hospital Analytics Dashboard | End-to-End Data Analytics Project (Power BI + Python)

An end-to-end healthcare analytics project transforming raw hospital data into actionable insights using data cleaning, modeling, and interactive dashboards. The solution enables performance tracking, financial monitoring, and operational optimization across hospital departments.

---

* Short Description / Purpose
  The CareMetrics Hospital Analytics Dashboard is designed to analyze hospital operations, patient trends, financial performance, and staff efficiency. It helps stakeholders make data-driven decisions by providing insights into admissions, revenue, workload distribution, and treatment patterns.

---

* Tech Stack

* Python (Pandas, NumPy) – Data cleaning & preprocessing

* Power BI Desktop – Interactive dashboard creation

* Power Query – Data transformation

* DAX (Data Analysis Expressions) – KPI calculations

* Excel – Data storage & modeling

* Jupyter Notebook – Development environment

---

* Data Source

* Source: Simulated hospital dataset

* Date Range: January 2021 – January 2026

* Total Records: ~90,000 visits

* Unique Patients: ~70,000

* Staff Data:

* 75 Doctors

* 50 Surgeons

* 350 Nurses

* Key Fields:

* Patient_ID, Age, Gender

* Department_Name

* Doctor_ID, Nurse_ID, Surgeon_ID

* Admission_Date, Discharge_Date

* Length_of_Stay

* Treatment_Cost, Revenue

* Payment_Status, Insurance

* Visit_Type (Emergency / Planned)

---

* Data Cleaning & Processing

* Removed duplicates and handled missing values

* Standardized categorical data (Gender, Department, Payment Status)

* Created derived columns: Age_Group, Length_of_Stay, Cost_Category, Revenue Contribution

* Fixed inconsistent entries and null values

* Built structured dataset for analysis

---

* Data Modeling

The dataset was transformed into a relational model (Star Schema)

* Fact Table:

* Fact_Visit – Contains all transactional hospital data

* Dimension Tables:

* Patient

* Doctor

* Nurse

* Surgeon

* Department

* Admission

* Billing

* Business Problem

* Solution

* Certain departments contribute higher revenue share

* Workload is unevenly distributed among doctors

* Long stay patients impact operational efficiency

* Pending payments highlight revenue leakage

* Patient inflow varies seasonally

* Efficient doctors maintain low LOS with high patient count
